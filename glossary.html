HTML Tags Description
a Anchor. Primarily used as a hypertext link. The link can be to another page, a part of a page or any other location on the web.
abbr An abbreviation.
aside Content that is related to, but separate from, the content surrounding it. Pull-quotes or snippets of related information in an article are examples of content that could be marked up with aside tags.
blockquote A block of text that is quoted from elsewhere.
body The main content area of an HTML document. You must use this element and it should be used just once. It should start immediately after the closing head tag and end directly before the closing html tag.
br Legitimate line breaks can occur in the likes of addresses or poetry — when they are a meaningful part of the content. They should not be used for purely stylistic reasons, such as replicating the look of a paragraph (when a p element would be appropriate). br has no content and therefore does not warrant a closing tag.
col One or more columns in a table. col elements are children of colgroup elements. col has no content and therefore does not warrant a closing tag.
caption If used, the caption element should be the first child of a table element, appearing before any tr, tbody, thead, or tfoot elements.
div Division — a generic container for a block of HTML. Lending no additional meaning, it is typically used as scaffolding to hang CSS on or for JavaScript to reach out to.
data Associates machine-readable data with content. This can be used in conjunction with microdata (see Global attributes) or manipulated by JavaScript or other data processors.
embed Plugin point — where an external, typically non-HTML, application or resource can be embedded in a page.
footer The footer of a page or section. This could contain content such as a postscript, appendix, or links to related pages, for example.
h1 - h6 Ranked headings, h1 being the top-level heading, and h6 being the lowest level heading.
head Where metadata — information about the document — is placed. It should be the first element inside an html element.
header The header of a page or section, containing introductory content or navigation.
iframe An inline frame, or nested browsing context — essentially, a web page embedded within a web page.
img Image. This could be a photograph or graph or any other meaningful pictorial content. It should not be used purely for presentation, where CSS, such as background-image is more appropriate.
li List item. Used in conjunction with ul or ol to make an unordered list or ordered list respectively.
link Defines a link to an external resource. It is most commonly used to link a CSS file to an HTML document.
main The main content of the body — content that is unique to the page, excluding content that might be repeated on multiple pages, such as navigation.
meta Metadata — information about the HTML document — that can not be covered by the title, style, link, base, or script elements.
nav A navigation area — a major group of links to to other pages, or sections of a page.
object External resource, treated as an image, a nested browsing context, or a plugin.
p A paragraph.
pre Preformatted text. All of the whitespace within this element is regarded as semantically relevant (whereas other elements will consider the likes of indentations or multiple consecutive spaces to be meaningless).
script An executable script, such as JavaScript.
section A general section of an HTML document. This could be used to split up an article, or to represent chapters, for example.
source Alternative media resource for video and audio elements. A browser should embed the first resource it can — if it gets befuddled with one, it can move on to the next in a list.
style Applies styles, typically CSS, at a page-level (as opposed to linking to an external CSS file). Typically placed inside the head element.
table The basis of a table, used to format tabular data.
ul Unordered list — a list that has no logical sequence. Used in conjunction with li, which defines the list’s items.

All descriptions of tags taken from htmldog.com
