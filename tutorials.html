My tutorial of the week is from LearnWebCode on youtube, he makes using flexboxes seem simple. For those of you who are new to the web development area, flex boxes are a neccessity in page design. With devices ranging from small touch screen phones to giant desktop PC monitors the way it is viewed should be optimised for everyone.

With the ability to use flex boxes properly, web design will become a cake walk. The basics of CSS and HTML can take you where you need to go but using modern and current conventions is the best practice and the best way to stay relevant in the ever changing environment which is web development.
